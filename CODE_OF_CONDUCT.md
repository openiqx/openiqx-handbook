# OpenIQX Code of Conduct

## Our Commitment

The OpenIQX community is committed to creating a welcoming, respectful, and collaborative environment for everyone.

We believe open source thrives when people from different backgrounds, experiences, and skill levels can work together safely and respectfully.

All contributors, maintainers, and participants in OpenIQX spaces are expected to follow this Code of Conduct.

---

## Our Values

The OpenIQX community values:

* Respectful communication
* Constructive collaboration
* Openness to learning and teaching
* Professional behavior
* Inclusivity and kindness toward all participants

We encourage experienced contributors to support newcomers and help them grow.

---

## Expected Behavior

Participants in the OpenIQX community should:

* Be respectful and considerate in communication
* Provide constructive feedback and guidance
* Be welcoming to beginners and new contributors
* Respect differing opinions and perspectives
* Focus on improving the project and helping others succeed
* Use inclusive and professional language

---

## Unacceptable Behavior

The following behaviors are not acceptable in the OpenIQX community:

* Harassment or intimidation
* Personal attacks or insults
* Discriminatory language or behavior
* Hate speech or offensive comments
* Trolling, deliberate disruption, or hostility
* Publishing private information without consent

These behaviors are not tolerated in any OpenIQX spaces, including:

* GitHub repositories
* Pull requests and issues
* Discussions and chat platforms
* Community events or sessions

---

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

If unacceptable behavior occurs, maintainers may take appropriate actions, including:

* Issuing a warning
* Removing comments or contributions
* Temporarily restricting participation
* Removing individuals from the community

Enforcement decisions will prioritize maintaining a respectful and safe environment.

---

## Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, please report it to the project maintainers.

Reports will be handled confidentially and respectfully.

When reporting an issue, include:

* A description of the incident
* Links or screenshots if available
* Any relevant context

Maintainers will review the report and take appropriate action.

---

## Scope

This Code of Conduct applies to all OpenIQX spaces, including:

* GitHub repositories
* Contribution sessions
* Community discussions
* Any official OpenIQX events or communications

---

## A Shared Responsibility

Maintaining a healthy community is a shared responsibility.

By participating in OpenIQX, you agree to follow these guidelines and help create a respectful and welcoming environment for everyone.
